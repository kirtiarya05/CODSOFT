# 📝 To-Do List App (Python)

A simple GUI-based To-Do List application built using Python 3.11 and Tkinter.  
It allows users to add, update, and delete tasks using a clean interface.

## Requirements
- Python 3.11
- Tkinter (comes pre-installed with Python)

## How to Run
Save the file as `todo.py`, open terminal in the project folder, and run:

python todo.py

If `python` doesn’t work, use:

py todo.py
