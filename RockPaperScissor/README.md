# Rock Paper Scissors - Tournament Edition

A modern, GUI-based Rock Paper Scissors game with score tracking and tournament mechanics. Built for the **Codsoft Internship - Task 4**.

## 🎮 Features
- **Modern GUI**: Sleek dark theme using Tkinter and custom styling.
- **Score Tracking**: Persistent scoreboard tracking the user vs. computer.
- **Tournament Mode**: First to 10 points wins the championship!
- **Interactive Feedback**: Visual icons and color-coded results for wins, losses, and ties.
- **Play Again**: Built-in reset functionality to start new sessions.

## 🛠️ Installation

1. Navigate to the project directory:
   ```bash
   cd Task4_RockPaperScissor
   ```
2. Run the application:
   ```bash
   python main.py
   ```

## 🕹️ Controls
- Click the **Rock**, **Paper**, or **Scissors** buttons to make your move.
- The computer will simultaneously make its choice.
- The scoreboard updates automatically.
- A popup will announce the champion once someone reaches 10 points.
